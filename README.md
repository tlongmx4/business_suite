# Business Suite Backend (FastAPI)

A backend API for a multi-business scheduling tool.  
Built with FastAPI, SQLAlchemy, and Pydantic.

## Features
- Users
- Businesses
- Appointments
- Conflict detection
- SQLAlchemy models + CRUD
- FastAPI routers + docs

## How to run
uvicorn scheduler.app:app --reload
Visit the docs: http://127.0.0.1:8000/docs


## Tech
- FastAPI
- SQLAlchemy
- Pydantic
- SQLite (dev)
